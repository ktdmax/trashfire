How verification works: Every badge has a verify code (bottom right). The code is a hash of your result data. Anyone can check it at trashfire.io/badge > Verify tab. A badge is only "verified" if a matching run exists in the public leaderboard. Unverified badges still show your score - they just haven't been confirmed yet.

Paste a badge verify code to check if it matches a submitted run.

Where to find the code: It's printed on the bottom right of every badge, starting with TF- followed by 12 hex characters.

What "verified" means: The code matches a run that was submitted as a PR to the TRASHFIRE repo and accepted by maintainers. The full result data, skill text, and canonical prompt hash are all public and reproducible.

What "unverified" means: The code doesn't match any submitted run. The badge may still be real (just not submitted yet), or it may be fabricated.
